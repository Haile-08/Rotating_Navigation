Blockchain

Blockchain seems complicated, and it definitely can be, but its core concept is
really quite simple. A blockchain is a type of database. To be able to understand
blockchain, it helps to first understand what a database actually is.

A database is a collection of information that is stored electronically on a
computer system. Information, or data, in databases is typically structured in table
format to allow for easier searching and filtering for specific information.
